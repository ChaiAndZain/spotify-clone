# Spotify Clone

A simple web-based clone of the Spotify music player interface, built using vanilla HTML, CSS, and JavaScript.

## Demo
[Live Demo](https://spotify-clone-7ca84.web.app)



## Features
- Functional audio player with play, pause, and seek capabilities.
- Responsive design for various screen sizes.
- Clean and modern user interface mimicking Spotify.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository.
2. Open `index.html` in your web browser.

## Project Structure
- `index.html`: Main HTML structure.
- `style.css`: Styling and layout.
- `dom_script.js`: Handles DOM manipulation
- `player_script.js`: Manages audio playback logic.
- `public/`: Contains assets like images, songs and icons
